Project One: Self Checkout

Basic self-checkout system where values are read from a .txt file and disaplyed on screen. Users can double click items to add to their cart and checkout. It also includes an interactive manager menu where employees can manipulate the database while the program is running. This application was built in the Windows Presentation Foundation using C# and XAML.
