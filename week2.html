FB Friend Database Week 2 (05/24-05/30)

See the Pen Week 2 by Marek Sautter (@msautter) on CodePen.

Project Description

Originally, I developed this project in python and SQLite but I wanted to recreate it in Javascript. I also tried to integrate the Facebook API to generate a list of friends, but Facebook disabled this feature a few years ago.

Steps

  1. Go to your facebook friends list on your profile.
  2. Scroll down until all your friends have been loaded on the screen.
  3. Select all (ctrl+a) and paste (ctrl+v) into the text input to the left.
