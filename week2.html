FB Friend Database Week 2 (05/24-05/30)

Name Num Mut

Project Description

Originally, I developed this project in python and SQLite. I hope to come back to this project one day and maybe have a download link to a .db file with the user's friends.

Originally I tried to use the Facebook API, however due to security reasons, we are no longer able to access our lists of friends.

Steps

  1. Go to Facebook and go to your friends list.
  2. Scroll down until all your friends have been loaded on the screen.
  3. Select all (ctrl+a) and paste (ctrl+v) it into the text input to the left.
